


Trygstad Gård – WR2 Prosjekt

Dette prosjektet er laget som en del av Work Requirement 2. Målet var å lage en enkel nettside med HTML, CSS, navigasjon, skjema og riktig bruk av Git og GitHub.



Nettsiden består av:

index.html – hovedside med kategorier og banner
trefjottverk.html – underside
keramikk.html – underside
trygstad.html – underside
kontakt.html – kontaktskjema
takk.html – bekreftelsesside
style.css – styling
images/ – bilder brukt på siden



Funksjonalitet

Navigasjonsmeny på alle sider
Klikkbare bilder på forsiden som leder til undersider
“Kontakt oss”-knapp nederst på hver side
Skjema med HTML5‑validering
Takk‑side som bekrefter innsending
Teknologier brukt
HTML (semantic struktur)
CSS (Flexbox, layout, styling)
Git og GitHub for versjonskontroll
